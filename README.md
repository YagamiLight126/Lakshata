# 简单前端脚手架 Lakshata

## 概述
一个简单的前端项目脚手架，用于生成常用的前端模板代码

## 使用
```
npm install -g lakshata
lakshata init  // 输入有效模板名字中的一个

// 因为对应的模板也是以仓库的形式存在github上的的，所以进入对应项目后记得重新设置远端仓库地址为自己的项目
cd 'your_project' 
git remote add origin '你的项目的git地址'
```

## 有效的模板
- react-ts
- react-js

## Commands
- list        显示所有模板
- init        用一个模板生成项目
